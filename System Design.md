# User Roles
## Librarian
1. Manage Catalouge of Books.
2. Track Book loans.
3. Handle member registratioin.
4. Add,Update,and Remove books from Catalouge.
5. Oversee Check-ins.
6. Oversee Check-outs.
7. Manage fines.
8. Generate detailed reports.
   
## Member
1. Register account.
2. View profile.
3. Update profile.
4. access current loans.
5. access borrowing history.
6. Search for books.
7. Reserve available books.
